PATH

理论优先路径

L0Agent 基础理论:建立共同语义，完成后进入基础理论检查点。
L1九层架构总览:理解在线主链路、离线评测旁路以及组件职责,完成后进入九层架构检查点。
L2编排控制总览:进入五单元理论课程并完成无代码门禁。
L2推理内核课程:编排理论之后学习四个决策单元，再完成推理内核理论检查点。
L2工具 / 动作层课程:推理内核之后学习五个执行与可靠性单元，再完成工具理论检查点。
L2知识 / RAG 课程:工具理论之后学习五个外部知识检索单元，再完成知识 / RAG 理论检查点。
L2记忆系统课程:知识 / RAG 之后学习四个经历沉淀与治理单元，再完成记忆系统理论检查点。
L3最小 Agent Loop:理论通过后运行实验，再完成工程考核。
L4面试表达:用主回答、追问链和系统设计整合已掌握内容。
