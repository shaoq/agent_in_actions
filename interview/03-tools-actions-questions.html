工具 / 动作层面试题

从工具契约到副作用控制，讲清“模型提议、策略授权、执行器落地”。

题库总览工具课程建议答案
安全分布：在线授权在治理层，离线安全回归在评测层，代码和浏览器执行隔离在基础设施沙箱。
TOOL-01基础

怎样设计稳定的 Tool Contract？

考察目标：覆盖名称、schema、语义、错误和副作用。

对应课程：工具契约与 Schema

主问题：一个生产工具定义除了 JSON Schema 还需要什么？

连续追问：

  1. 字段描述怎样减少模型误用？
  2. 工具版本升级如何兼容旧 run？
  3. 业务错误应该怎样结构化返回？

练习要求：给出输入、输出、错误、权限和幂等契约。

TOOL-03进阶

TOOL-04进阶

对应课程：权限、审批与沙箱

主问题：代码执行或 Computer Use 为什么需要沙箱？

连续追问：

  1. 网络、文件、凭据应如何限制？

  3. 多租户执行如何避免数据串扰？

练习要求：覆盖资源配额、网络 allowlist、临时凭据和销毁。
